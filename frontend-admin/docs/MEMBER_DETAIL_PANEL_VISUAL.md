## Color System

### Status Colors
```
● Active:    Green  (#22c55e / bg-green-500)
● Suspended: Red    (#ef4444 / bg-red-500)
● Pending:   Yellow (#eab308 / bg-yellow-500)
● Inactive:  Gray   (#6b7280 / bg-gray-500)
```

### Risk Colors
```
● Low:      Green  (#22c55e)
● Medium:   Yellow (#eab308)
● High:     Orange (#f97316)
● Critical: Red    (#ef4444)
```

### Tier Colors
```
● Basic:   Gray   (#6b7280)
● Premium: Blue   (#3b82f6)
● VIP:     Purple (#a855f7)
```

### Verification Colors
```
● Verified: Green  (#22c55e)
● Pending:  Yellow (#eab308)
● Rejected: Red    (#ef4444)
● None:     Gray   (#6b7280)
```

## Animation Timings

```
Panel Slide-in:    300ms spring (damping: 30, stiffness: 300)
Modal Fade-in:     200ms ease-out
Tab Switch:        150ms ease-in-out
Hover Scale:       200ms ease-out (scale: 1.02)
Button Press:      100ms ease-out (scale: 0.98)
Timeline Stagger:  50ms delay per item
Risk Collapse:     300ms ease-in-out
```

## Responsive Breakpoints

```
Desktop:  ≥ 768px  →  520px panel width, 2-column layout
Tablet:   < 768px  →  Full-screen drawer, stacked layout
Mobile:   < 640px  →  Full-screen, single column, bottom sheets
```

## Dark Mode Support

All components include dark mode variants:
- Background: `bg-white dark:bg-gray-900`
- Text: `text-gray-900 dark:text-white`
- Borders: `border-gray-200 dark:border-gray-700`
- Cards: `bg-gray-50 dark:bg-gray-800`
